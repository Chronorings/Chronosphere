# Chronosphere

## 【概要】
これは、**時環論（Chronorings）** における認知モデルを簡単に可視化するシステムです。

当該システムでは、デフォルトとして「秒の時環」「分の時環」「時間の時環」「日の時環」の４種類を用意していますが、ユーザーが自由に時環を追加/編集/削除できます。

マニュアルなしでも直感的に操作できるよう設計されていますが、以下に操作方法を記載いたします。

## 【操作方法】
### 時環について
- 前述の通り、デフォルトで「秒の時環」「分の時環」「時間の時環」「日の時環」の４種類を用意しています。
- 時環にカーソルを合わせると、背景に時環の名前が表示されます。
- 時環を左クリックすると、時環の名前と時間が確認でき、それぞれインライン編集ができます。
- 詳細パネルの下にあるDeleteから、時環を削除できます。
### 左下のUIパネルについて
- Scale：時環の大きさと間隔を変更します。
- All Notifications：全時環の通知の切り替えができます。
- Add Ring：時環を追加します。UI上部に、時環の名前と時間を入力するダイアログが表示されます。
- Delete All：全ての時環を削除します。UI上部に、確認画面が表示されます。
### 通知機能について
- 時環が１周すると、画面右上に「"名前" completed one cycle」という通知が表示されます。
- 前述の通り、左下のUIパネルか詳細パネルから通知のON/OFFを切り替えられます。
### 時間入力について
- 時間は「DD:HH:MM:SS」「?d?h?m?s」「??[s]」で入力できます。

例）１分半を入力したいとき -> 「00:00:01:30」「1m30s」「90(s)」（数字のみ入力した場合は、自動で秒数として入力されます。）
