# Chronosphere
## 【概要】
これは、**時環論（Chronorings）** の認知モデルを簡単に可視化するシステムです。

当該システムでは、デフォルトとして「秒の時環」「分の時環」「時間の時環」「日の時環」の４種類を用意していますが、ユーザーが自由に時環を追加/編集/削除できます。

マニュアルなしでも直感的に操作できるよう設計されていますが、以下に操作方法を記載いたします。

## 【操作方法】
### 時環について
- 前述の通り、デフォルトで「秒の時環」「分の時環」「時間の時環」「日の時環」の４種類を用意しています。
- 時環にカーソルを合わせると、背景に時環の名前が表示されます。
- 時環を左クリックすると、時環の詳細が確認できるほか、編集と削除が選択できます。
### 左下のUIパネルについて
- Scale：時環の大きさと間隔を変更します。
- Add Ring：時環を追加します。UI上部に、時環の名前と時間を入力するダイアログが表示されます。
- Delete All：全ての時環を削除します。UI上部に、確認画面が表示されます。
### 全操作系統
- 全てのダイアログの右上に、閉じるボタンがついています。
- 時間は「HH:MM:SS」「?h?m?s」「??[s]」で入力できます。

例）１分半を入力したいとき -> 「00:01:30」「1m30s」「30(s)」（数字のみ入力した場合は、自動で秒数として入力されます。）
